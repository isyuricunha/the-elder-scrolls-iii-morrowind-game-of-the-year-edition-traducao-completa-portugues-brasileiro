# The Elder Scrolls III: Morrowind GOTY - Tradução Completa PT-BR

Tradução completa de **The Elder Scrolls III: Morrowind - Game of the Year Edition** para português brasileiro.

Este pacote substitui os arquivos mestres oficiais do jogo:

- `Morrowind.esm`
- `Tribunal.esm`
- `Bloodmoon.esm`

Por isso, antes de instalar, faça backup dos arquivos originais.

## Requisitos

Você precisa ter o **The Elder Scrolls III: Morrowind - Game of the Year Edition** instalado no computador, com as expansões **Tribunal** e **Bloodmoon**.

Também precisa de um programa para extrair arquivos compactados:

- ZIP: o próprio Windows consegue extrair.
- 7Z: use o 7-Zip ou outro programa compatível.

## Onde instalar

A tradução deve ser instalada na pasta `Data Files` do Morrowind. No meu caso, o caminho usado é:

```text
D:\Morrowind\Data Files
```

Esse caminho pode mudar dependendo de onde você instalou o jogo. Alguns caminhos comuns no Windows:

```text
C:\Program Files (x86)\Steam\steamapps\common\Morrowind\Data Files
C:\Program Files\Steam\SteamApps\common\Morrowind\Data Files
C:\Program Files (x86)\GOG Galaxy\Games\Morrowind\Data Files
C:\GOG Games\Morrowind\Data Files
C:\Program Files\Bethesda Softworks\Morrowind\Data Files
```

Se você não souber onde o jogo está instalado, procure pela pasta que contém `Morrowind.exe` e, dentro dela, abra a pasta `Data Files`.

Na Steam, você também pode clicar com o botão direito no jogo, abrir **Gerenciar > Procurar arquivos locais** e depois entrar em `Data Files`.

## Como instalar a tradução

### Passo 1: baixar os arquivos

1. Abra a página de downloads ou releases:
   NexusMod: <https://www.nexusmods.com/morrowind/mods/59287?tab=files>
   GitHub: <https://github.com/isyuricunha/the-elder-scrolls-iii-morrowind-game-of-the-year-edition-traducao-completa-portugues-brasileiro/releases/latest>
2. Baixe o arquivo `traducao.zip` ou `traducao.7z`.
3. Salve o arquivo em um lugar fácil de encontrar, como `Downloads` ou `Desktop`.

Também dá para baixar o repositório pelo botão **Code > Download ZIP**, mas a opção de **Releases** é a mais recomendada porque os arquivos já ficam empacotados para instalação.

### Passo 2: extrair o pacote

1. Clique com o botão direito no arquivo baixado.
2. Escolha **Extrair Tudo...** se for ZIP, ou use o 7-Zip se for 7Z.
3. Depois de extrair, abra a pasta da tradução.
4. Você deve encontrar estes arquivos:
   - `Morrowind.esm`
   - `Tribunal.esm`
   - `Bloodmoon.esm`

Se os arquivos estiverem dentro de uma pasta chamada `traducao`, abra essa pasta.

### Passo 3: fazer backup dos arquivos originais

Antes de substituir qualquer coisa, faça backup dos arquivos originais do jogo.

1. Abra a pasta `Data Files` do Morrowind.
2. Encontre estes arquivos:
   - `Morrowind.esm`
   - `Tribunal.esm`
   - `Bloodmoon.esm`
3. Crie uma pasta chamada `backup-original`.
4. Copie os três arquivos originais para dentro de `backup-original`.

Não pule esta etapa. Como a tradução substitui arquivos oficiais, o backup permite voltar ao jogo original sem precisar reinstalar tudo.

### Passo 4: copiar a tradução para o jogo

1. Copie os três arquivos traduzidos:
   - `Morrowind.esm`
   - `Tribunal.esm`
   - `Bloodmoon.esm`
2. Cole na pasta `Data Files` do Morrowind.
3. Quando o Windows perguntar se deseja substituir os arquivos, confirme a substituição.

Exemplo de destino:

```text
D:\Morrowind\Data Files
```

Depois da cópia, a pasta `Data Files` deve conter os três arquivos `.esm` traduzidos no lugar dos originais.

### Passo 5: abrir o jogo

Abra o Morrowind normalmente. Não precisa ativar mod em launcher nem usar gerenciador de mods.

Se tudo foi copiado corretamente, o jogo já deve carregar os textos em português brasileiro.

## Como restaurar os arquivos originais

Se quiser remover a tradução:

1. Abra a pasta `Data Files`.
2. Copie os arquivos guardados em `backup-original`.
3. Cole de volta em `Data Files`.
4. Confirme a substituição.

Outra opção é usar a verificação de arquivos da loja onde você comprou o jogo, como Steam ou GOG Galaxy. Isso baixa novamente os arquivos oficiais, mas pode desfazer outras alterações instaladas no jogo.

## Solução de problemas

### O jogo continua em inglês

Verifique se você substituiu os três arquivos dentro da pasta `Data Files` correta. Se o jogo estiver instalado em mais de um lugar, pode ser que a tradução tenha sido copiada para a pasta errada.

### O Windows não deixa substituir os arquivos

Feche o jogo e o launcher antes de copiar os arquivos. Se o jogo estiver instalado dentro de `Program Files`, talvez o Windows peça permissão de administrador.

### Não encontrei a pasta Data Files

Procure no computador por `Morrowind.esm`. A pasta onde esse arquivo estiver é a pasta correta para instalar a tradução.

### Uso OpenMW

Use a mesma pasta `Data Files` que o OpenMW está usando para carregar o jogo. Se você configurou uma cópia separada dos dados do Morrowind no OpenMW, substitua os `.esm` nessa cópia.
